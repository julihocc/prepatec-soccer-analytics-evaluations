# Rúbricas de Evaluación - Bloque 2

## Información General

Este directorio contiene las rúbricas detalladas para evaluar todas las actividades del Bloque 2: Fundamentos de Ciencia de Datos aplicados al análisis deportivo.

## Estructura de Evaluación del Bloque 2

### Distribución de Calificaciones
- **Ejercicios Semanales (Semanas 6-10)**: 30% del bloque
- **Proyecto de Análisis Principal**: 70% del bloque
- **Peso en curso total**: 35% de la calificación final

## Criterios Específicos de Retroalimentación

### Retroalimentación Formativa

#### Para Nivel Sobresaliente (9.5-10.0)
- **Fortalezas identificadas:** Aspectos excepcionales del trabajo
- **Extensiones sugeridas:** Direcciones para profundización adicional
- **Aplicaciones avanzadas:** Oportunidades para aplicar conocimientos en contextos más complejos

#### Para Nivel Competente (7.0-9.4)
- **Logros reconocidos:** Aspectos bien ejecutados
- **Áreas de mejora:** Aspectos específicos para perfeccionar
- **Próximos pasos:** Sugerencias para desarrollo continuo

#### Para Nivel Satisfactorio (6.0-6.9)
- **Progreso reconocido:** Aspectos que muestran comprensión básica
- **Refuerzo necesario:** Conceptos que requieren práctica adicional
- **Recursos de apoyo:** Materiales y actividades de refuerzo específicas

#### Para Nivel Insuficiente (0-5.9)
- **Identificación de gaps:** Conceptos fundamentales no comprendidos
- **Plan de recuperación:** Pasos específicos para alcanzar competencia
- **Soporte intensivo:** Recursos y apoyo adicional requerido

## Consideraciones Especiales

### Adaptaciones para Diferentes Estilos de Aprendizaje
- **Visual:** Valoración especial de calidad de visualizaciones
- **Analítico:** Reconocimiento de rigor metodológico
- **Práctico:** Énfasis en aplicabilidad de análisis
- **Creativo:** Valoración de enfoques innovadores

### Factores de Contexto
- **Tiempo disponible:** Ajustes por limitaciones temporales justificadas
- **Recursos técnicos:** Consideración de limitaciones de hardware/software
- **Nivel previo:** Reconocimiento de progreso individual desde punto de partida
- **Intereses específicos:** Valoración de conexiones con intereses personales en el deporte

### Oportunidades de Mejora
- **Resubmisión parcial:** Posibilidad de mejorar aspectos específicos
- **Trabajo adicional:** Oportunidades de práctica extra
- **Mentoría:** Sesiones individuales de apoyo
- **Colaboración peer:** Aprendizaje entre compañeros

---

**Las rúbricas están diseñadas para ser herramientas de aprendizaje que guíen el desarrollo de competencias en análisis de datos deportivos mientras mantienen estándares académicos apropiados para el nivel preparatoria.**
