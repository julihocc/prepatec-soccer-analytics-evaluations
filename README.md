# Sistema de Evaluaciones - Ciencia de Datos Aplicada al Fútbol

## 📋 Información General

Este directorio contiene todas las evaluaciones del curso "Ciencia de Datos Aplicada al Análisis de Resultados en Fútbol", organizadas por bloques y diseñadas para evaluar progresivamente las competencias adquiridas.

## 🎯 Filosofía de Evaluación

### Evaluación Formativa y Continua
- **Enfoque progresivo**: Cada evaluación construye sobre la anterior
- **Aplicación práctica**: Todos los ejercicios están contextualizados en análisis deportivo
- **Aprendizaje activo**: Las evaluaciones son herramientas de aprendizaje, no solo de medición

### Criterios de Evaluación
1. **Correctitud técnica** (30%): Código funcional y uso apropiado de herramientas
2. **Calidad del análisis** (25%): Profundidad y rigor en el análisis de datos
3. **Presentación y comunicación** (25%): Claridad en visualizaciones y explicaciones
4. **Aplicación contextual** (20%): Relevancia y pertinencia deportiva del análisis

## 📚 Estructura de Evaluaciones

**Semanas 11-15 | Mi Primera Predicción**

## 🎯 Competencias Evaluadas

### **Bloque 1: Competencias de Programación**
- **Sintaxis de Python**: Variables, tipos de datos, operadores
- **Estructuras de control**: Condicionales, bucles, funciones
- **Estructuras de datos**: Listas, diccionarios, manejo básico
- **Librerías básicas**: pandas, numpy, matplotlib (nivel introductorio)

### **Bloque 2: Competencias de Análisis**
- **Exploración de datos**: Carga, limpieza, exploración inicial
- **Estadística descriptiva**: Medidas de tendencia central y dispersión
- **Visualización**: Gráficos efectivos con seaborn y matplotlib
- **Interpretación**: Conclusiones basadas en evidencia estadística

### **Bloque 3: Competencias Predictivas**
- **Modelos básicos**: Regresión logística, Random Forest
- **Evaluación de modelos**: Precisión, matrices de confusión
- **Interpretación de resultados**: Comprensión de predicciones
- **Aplicación práctica**: Solución de problemas deportivos reales

## 🏆 Criterios de Éxito

### **Nivel Satisfactorio (7.0 - 8.4)**
- Demuestra comprensión básica de conceptos
- Código funcional con errores menores
- Análisis correcto pero básico
- Comunicación clara de resultados

### **Nivel Competente (8.5 - 9.4)**
- Dominio sólido de conceptos y herramientas
- Código limpio y bien estructurado
- Análisis profundo con insights relevantes
- Excelente comunicación y visualización

### **Nivel Sobresaliente (9.5 - 10.0)**
- Aplicación creativa y avanzada de conceptos
- Código optimizado y bien documentado
- Análisis innovador con conclusiones valiosas
- Presentación profesional y insights únicos
